# Asset Placeholders

This directory contains placeholder assets for development.

## Required Files

The following placeholder files have been created to prevent build errors:

- **icon.png** (1024x1024) - App icon
- **adaptive-icon.png** (1024x1024) - Android adaptive icon
- **splash.png** (1284x2778) - Splash screen
- **favicon.png** (48x48) - Web favicon

## To Replace Placeholders

1. Create proper branded assets with:
   - Purple gradient background (#1a0933)
   - "فقش" logo in white
   - Matching web app branding

2. Replace the placeholder files in this directory

3. Restart Expo dev server

## Image Specifications

### icon.png
- Size: 1024x1024px
- Format: PNG with transparency
- Used for: iOS and Android app icon

### adaptive-icon.png
- Size: 1024x1024px
- Format: PNG with transparency
- Used for: Android adaptive icon (foreground layer)
- Safe area: Center 512x512px

### splash.png
- Size: 1284x2778px (iPhone 13 Pro Max)
- Format: PNG
- Safe area: 1170x2532px centered
- Background: #1a0933 (matching app.json)

### favicon.png
- Size: 48x48px
- Format: PNG
- Used for: Web browser tab icon
